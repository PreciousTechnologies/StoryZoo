# Story Zoo Assets

## Directory Structure

### /images
Place story cover images, logos, and illustrations here.

### /icons
Place app icons and custom icon assets here.

### /audio
Place audio narration files here.

### /fonts
Place custom font files here (Poppins recommended).

## Image Guidelines

- Use high-quality images (minimum 1080px width for covers)
- Supported formats: PNG, JPG, WebP
- Optimize images for mobile (compress without losing quality)

## Audio Guidelines

- Format: MP3 or AAC
- Quality: 128kbps minimum
- Mono audio for speech is acceptable
- Include metadata (title, artist)
