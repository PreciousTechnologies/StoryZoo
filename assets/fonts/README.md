# Font Files

To complete the app setup, add Poppins font files to this directory:

- Poppins-Regular.ttf
- Poppins-Medium.ttf
- Poppins-SemiBold.ttf
- Poppins-Bold.ttf

You can download Poppins font from:
- Google Fonts: https://fonts.google.com/specimen/Poppins
- Or use any other preferred font family

Note: If you don't add fonts, the app will use the system default font.
